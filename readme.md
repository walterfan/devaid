Developer Aid Toolkit 1.0
-----------------------------
* Encode conversion Tool

* Regular Expression Tool

* SQL Tool

* Http Tool

# Prerequisites
* Install JDK 1.6 or above
* Install Maven 2 or above

# Execution
## Steps
* 1) mvn package
* 2) java -jar ./target/devaid-1.0-jar-with-dependencies.jar
